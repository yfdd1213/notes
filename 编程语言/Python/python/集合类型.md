##### 集合特性
- 集合元素之间无序，每个元素唯一，不存在相同元素
- 集合元素不可更改，不可变数据类型
- [[集合方法]]
##### 创建集合
- 大括号
- [[set()]]
- 集合生成式
```python
# 大括号
a = {1,2,3,4,5}
a = {'1','p','2','3','y'}

# set()
a = set('pypy123')

# 集合生成式
# 表达式接for循环
{i**2 for i in (1,2,3)}  # {1, 4, 9}
# 表达式接for循环，接if判断
{x for x in 'abracadabra' if x not in 'abc'}  # {'d', 'r'}
```
