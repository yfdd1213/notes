- `len( s )`
	- 返回对象（字符、列表、元组等）长度或项目个数
```python
string = "Hello, world!"
length = len(string)
print(length)  # 输出：13

numbers = [1, 2, 3, 4, 5]
length = len(numbers)
print(length)  # 输出：5

my_tuple = (2.5, 3.5, 4.5)
length = len(my_tuple)
print(length)  # 输出：3

my_set = {10, 20, 30}
length = len(my_set)
print(length)  # 输出：3

my_dict = {'a': 1, 'b': 2, 'c': 3}
length = len(my_dict)
print(length)  # 输出：3

```
