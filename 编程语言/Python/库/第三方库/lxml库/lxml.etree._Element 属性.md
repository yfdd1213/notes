##### lxml.etree._Element 属性
```python
_Element.tag  # 标签名
_Element.text  # 第一个子元素之前的文本。如果没有文本，这要么是字符串，要么是值None。
_Element.base  # 元素的基本URI
_Element.nsmap  # 命名空间前缀
_Element.prefix  # 命名空间前缀
_Element.sourceline  # 解析器找到的原始行号，如果未知则为“无”。
_Element.tail  # 文本位于此元素的结束标记之后，但位于下一个同级元素的开始标记之前。如果没有文本，这要么是字符串，要么是值None。
```
